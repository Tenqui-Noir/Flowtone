# Test Checklist

## 核心功能:

1. 首次打开 App，权限请求正常
2. 授权后能扫描出本地歌曲
3. 点击歌曲能播放
4. MiniPlayer 出现
5. MiniPlayer 显示标题、艺术家、封面
6. 暂停 / 继续正常
7. 下一首正常
8. 上一首正常
9. 拖动进度条正常
10. 播放完一首后能自动进入下一首
11. 队列末尾播放结束后停止，不循环
12. 返回桌面后继续播放
13. 从最近任务划掉 App 后，按当前设计确认播放行为是否正常
14. 系统媒体控件暂停 / 继续正常
15. 系统媒体控件上一首 / 下一首正常

### UI 测试：

1. 无歌曲时 MiniPlayer 不显示或按设计隐藏
2. 有歌曲时 MiniPlayer 贴底显示正常
3. 点击 / 上滑展开正常
4. 展开高度正常
5. 封面动画正常
6. 背景模糊 / 流云背景正常
7. 进度条位置和拖动手感正常
8. 三个控制按钮位置正常
9. 暂停 / 播放按钮状态同步正常
10. 深色模式文字可读
11. 浅色模式文字可读
12. 横竖屏切换后不崩，如果当前不支持横屏，也确认行为符合预期

### 播放层测试：

1. App 内点击播放
2. App 内暂停 / 继续
3. App 内上一首 / 下一首
4. 系统通知栏暂停 / 继续
5. 系统通知栏上一首 / 下一首
6. 蓝牙耳机 / 线控暂停继续，如果有设备
7. 锁屏媒体控件显示歌曲信息
8. 锁屏媒体控件暂停 / 继续
9. 后台播放 1 分钟不被异常停止
10. 播放过程中切歌，系统控件标题同步
11. 播放过程中暂停，系统控件状态同步

### 数据层测试：

1. 扫描歌曲数量正常
2. 歌曲标题正常
3. 艺术家正常
4. 时长正常
5. 封面正常
6. 点击第 1 首播放正常
7. 点击中间某首播放正常
8. 点击最后一首播放正常
9. 播放队列顺序正常
10. 播放状态恢复正常
